ResultsReporter

ResultsReporter class Null safety

This class logs the mutations and can create report documents in different formats. See ReportFormat for the supported reports.

It may create multiple folders in the output path depending on the selected report format.

Constructors

ResultsReporter(String inputFile, bool builtinRulesAdded)
Creates a test runner and adds inputFile to the xml input file list. builtinRulesAdded sets the flag in the report file when the builtin rules were added.
