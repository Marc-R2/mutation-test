Runs the mutation tests using the inputs from inputs. Undetected modifications are written to a file in outputPath using the specified format.

The testrunner will use builtin mutation rules if builtinRules is set to true. Additionally the ruleFiles will be loaded.

The amount of output to the command line is controlled via verbose. You can perform a dry run that wil not run any tests or perform any modifications, but will list all found mutations per file. Returns true if all modifications were detected by the test commands.

Writing output to the command line can be suppressed if quiet is set.
